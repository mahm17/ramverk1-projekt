Install the module
----------------

```
composer require mahm/projekt
```

Copy the config files
--------

```
rsync -av vendor/mahm/projekt/config ./
```

Copy the view files
--------------------

```
rsync -av vendor/mahm/projet/view ./
```

Copy the src files
--------------------

```
rsync -av vendor/mahm/projekt/src ./
```
